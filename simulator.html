🛡️ About This Toolkit

Defensive Python Labs teaches real-world cybersecurity concepts through hands-on Python code. Every module is built with zero required dependencies and runs from a single CLI entry point.

📦 Install

pip install defensive-python-labs

⚖️ Ethical Use

✅ Allowed

Your own systems, CTF challenges, penetration testing with written permission, academic research, security education.

🚫 Not Allowed

Scanning systems you don't own, unauthorized testing, any malicious use. This toolkit is for defenders, not attackers.
